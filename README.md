# Python-Airport-Flight-Data-Analysis
## Description
This project analyzes flight data from different airports using Python.
It processes CSV files, calculates statistics, and generates histograms.

## Features
- CSV file loading and validation
- Flight count analysis per airport and airline
- Histogram visualization of flight departures
- Simple data reporting in text file

## Technologies
- Python
- Graphics library
- CSV handling
- Basic data analysis
